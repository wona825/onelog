# ONELOG 🧑‍💻

**ONELOG**는 사람들에게 공유할 수 있는 개발자용 포트폴리오 웹사이트 프로젝트입니다.

<br>

## 프로젝트 설명
**ONELOG**는 개발자 포트폴리오를 보여주는 웹 애플리케이션입니다.
<br> 사용자는 직관적이고 간편하게 개발자의 다양한 정보를 열람하고, 궁금한 점을 챗봇에게 실시간으로 물어볼 수 있습니다.
<br> 현재 제 개발자 포트폴리오만 원페이지 형식으로 존재하나, 추후 포트폴리오 구성을 어려워하는 개발자들을 대상으로 프로젝트를 확장할 예정입니다. 

<br>

## 주요 기능
### 1. **자기소개 및 연락처 🙋**
- 개발자의 기본 정보와 연락처를 한눈에 볼 수 있습니다.
![image](https://github.com/wona825/onelog/assets/81519167/95da465a-3579-4244-8319-3135b611f99e)

### 2. **Skills 💻**
- 개발자가 보유한 기술 스택을 카테고리별로 나누어 확인할 수 있습니다. (예: Language, IDE & Tools, Framework, DBMS 등)
![image](https://github.com/wona825/onelog/assets/81519167/54e83871-b8c2-40ed-bf50-f047bb76f7fa)

### 3. **Careers 💼**
- 개발자의 경력 사항을 타임라인 형식으로 정리하여, 각 직장에서의 주요 업무와 성과를 확인할 수 있습니다.
  <img width="1078" alt="스크린샷 2024-06-12 오전 12 23 31" src="https://github.com/wona825/onelog/assets/81519167/fd700a54-184b-48f5-84cc-a58597ef2a6b">

### 4. **Projects 📄**
- 개발자가 진행한 프로젝트를 웹, 모바일, 기타 카테고리로 나누어 소개합니다.
- 프로젝트를 클릭하면, 해당 프로젝트의 introduction, main function, position, used skills, link 등의 상세 내용을 볼 수 있습니다. 
![image](https://github.com/wona825/onelog/assets/81519167/b53a6a88-76f0-489e-8c15-c75b6ef5df47)
  <img width="1309" alt="스크린샷 2024-06-12 오전 12 20 48" src="https://github.com/wona825/onelog/assets/81519167/7ccad723-bc1e-4d05-a832-ea220b04340a">


### 5. **챗봇 💬**
- 우측 하단의 챗봇에게 사용자가 궁금한 점을 물어보면 GPT API를 통해 해당 개발자의 정보를 기반으로 실시간으로 답변을 제공합니다.
  <img width="1290" alt="스크린샷 2024-06-12 오전 12 28 05" src="https://github.com/wona825/onelog/assets/81519167/ad016b3a-2f1f-4e2b-b191-3fde71f4fb58">


### 6. **Question Log ❓**
- footer 부분에는 다른 접속자들이 오늘 챗봇에게 한 질문들을 모아 볼 수 있고, 이를 통해 사용자는 다른 사람들이 궁금해했던 점들을 확인할 수 있습니다.
